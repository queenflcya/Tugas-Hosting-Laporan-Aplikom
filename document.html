LAPORAN PRAKTIKUM APLIKASI KOMPUTER

Hardware

Hardware adalah segala piranti atau komponen dari sebuah komputer yang sifatnya bisa dilihat secara kasat mata dan bisa diraba secara langsung. Hardware atau perangkat keras berfungsi untuk mendukung segala proses dalam komputerisasi dan bekerja sesuai perintah yang sudah ditentukan penggunanya atau yang sering disebut dengan brainware.

Sistem Operasi Windows

Sistem Operasi Windows adalah sistem operasi yang dikembangkan oleh Microsoft Corporation yang menggunakan antarmuka dengan berbasiskan GUI(Graphical User Interface) atau tampilan antarmuka bergrafis.

Sistem Operasi Linux

Sistem Operasi Linux adalah perangkat lunak atau software sistem operasi yang sifatnya open source dan gratis untuk didapatkan maupun disebarluaskan dengan lisensi GNU. OS Linux merupakan turunan dari unix dan dapat digunakan pada bermacam-macam komputer.

Microsoft Office Word

Microsoft Office Word adalah software (perangkat lunak) yang berfungsi untuk mengolah huruf-huruf atau kata-kata.Microsoft Office Word termasuk aplikasi pengolah kata yang sangat populer saat ini, aplikasi yang dikembangkan oleh Microsoft ini terdapat dalam satu paket Microsoft Office yang berisi Microsoft Word, Microsoft Excel, Microsoft Power Point, Microsoft Office Publisher, Microsoft Office Access, Microsoft Outlook, Microsoft Visio, Microsoft OneNote, Microsoft SharePoint, Microsoft InfoPath .

Microsoft Office Excel dan PowerPoint

Microsoft Excel adalah salah satu dari sekian banyak program aplikasi yang terdapat dalam paket aplikasi perkantoran Microsoft Office yan dikhususkan untuk pengolahan kata.

Desain Grafis

Desain grafis adalah proses komunikasi menggunakan elemen visual, seperti tulisan, bentuk, dan gambar yang dimaksudkan untuk menciptakan persepsi akan suatu pesan yang disampaikan.

Multimedia (Editing Video)

Multimedia adalah suatu sarana atau media melalui penggunaan komputer dalam atau untuk menggabungkan dan menyajikan suara, teks, animasi, audio dan video dengan alat bantu dan koneksi sehingga pengguna dapat bernavigasi, berinteraksi,berkarya dan berkomunikasi.

Pemrograman Web (HTML/CSS)

HTML adalah bahasa pemrograman standar yang digunakan untuk membuat sebuah halaman web, yang kemudian dapat diakses untuk menampilkan berbagai informasi di dalam sebuah penjelajahan web internet(Browser).

Pemrograman Web (Web Hosting)

Hosting adalah layanan jasa atau penyewaan tempat untuk menyimpan file atau bentuk script yang berada di internet dan memungkinkan untuk perorangan atau pun organisasi guna mnampilkan layanan jasa atau produk di web atau bahkan situs internet seperti web portal, web pribadi dan banyak lagi.

Augmented Reality (Blender)

Augmented Reality adalah teknologi yang memperluas dunia fisik dengan cara menambahkan lapisan informasi digital ke dalamannya.
